## ⚠️ How to Use .reg Files Safely

1. **Backup your Registry first:**  
   `Win + R` → type `regedit` → File → Export → save somewhere safe

2. **Double-click** the .reg file

3. Click **Yes** on the UAC prompt (admin required)

4. Click **Yes** on the Registry confirmation

5. **Restart your PC**

> These files only add or remove specific policy keys. They do not touch anything else in your system.
